# Focus Flow

A single-file Pomodoro-style focus timer with task tracking, daily streaks, and session history — all stored locally in the browser.

## Features

- **Focus / Break / Long Break timer** with a circular progress ring
- **Task list** — add tasks, mark them active, track pomodoros completed per task
- **Daily streaks** and session counts, persisted via `localStorage`
- **Completion overlay** with confetti when a session finishes

## Usage

Open `index.html` in any modern browser — no build step, server, or dependencies required.
